【万能論の終焉】Googleも撤退
ニューヨークのテック界隈では、GoogleのようなAIのトップランナーが、鳴り物入りのプロジェクトをシャットダウンしたことに衝撃が走っています。Googleがウェブ上でのタスク実行を目的とした実験的AI機能「Project Mariner」の撤退を発表したことは、AIが「魔法の杖」ではないという現実を、最も声高に喧伝してきた企業自身が認めたようなものです。Googleの「AIファースト」戦略への疑問符がつき始めているのは当然の流れでしょう。 Project Marinerは、ウェブ上の多様なタスクをAIが自律的に実行するという野心的な構想でした。しかし、ウェブ環境の複雑さ、多様なユーザーインターフェース、予期せぬエラーへの対応、そして何よりも「信頼性」の確保が極めて困難だったと見られます。AIが人間のように完璧にタスクをこなすことは、現時点では幻想だったのです。MicrosoftがXbox向けAIアシスタント「Xbox Copilot」の開発を中止したこと（過去記事【AI万能論】MSが撤退！）や、SnapとPerplexityの巨額提携が解消されたこと（過去記事【泡の終焉】Snap破談）と軌を一にする動きです。AIの適用範囲には、明確な限界があるという共通認識が形成されつつあります。 シリコンバレーのAI研究者たちは、AIの汎用性にはまだ大きな隔たりがあることを以前から指摘していました。特定のタスクに特化したAIは進化していますが、人間のように「常識」に基づいて多様な環境で自律的に動くAIは、想像以上に遠い未来の話です。ウォール街は、大手テック企業が過剰なAI投資を整理し始める兆候と見ています。AIブームの初期段階では、何でもAIで解決できるという楽観論が支配的でしたが、実際にプロダクトとして提供しようとすると、技術的な課題だけでなく、コスト、倫理、安全性など、乗り越えなければならない壁が次々と現れる。企業は「AIを使っていること」自体が目的化するフェーズから、真の価値提供を追求するフェーズへの転換を迫られています。
